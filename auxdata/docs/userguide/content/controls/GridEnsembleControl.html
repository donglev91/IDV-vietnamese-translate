{Ensemble Grid Control} { Ensembles capture samples of possible forecast scenarios. [ht::cimg images/GridEnsembleControlGui.gif] By default, the IDV displays all ensemble members. In the Field Selector, Displays panel, the user can subset the available ensembles in the Ensemble tab. The user may select a range of ensembles with the shift mouse button. The user can also select multiple ensemble members with meta-key mouse button. [ht::cimg images/GridEnsembleControlGui1.gif] In the Dashboard's Displays tab, the user can disable "Color by Member". The user will then have the capability of choosing an appropriate color table and color range for their data. [ht::cimg images/GridEnsembleControlGui2.gif] }
