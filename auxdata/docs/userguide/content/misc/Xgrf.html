[ug::head {XGRF Symbols}] The drawing control can export and import its drawings in an xgrf format. In this format you can specify a symbol type. The symbol tag in the xgrf has a symbol attribute, the value of which can be any of the below symbols.

    • FILLED_CIRCLE - Filled Circle

    • FILLED_DIAMOND - Filled Diamond
