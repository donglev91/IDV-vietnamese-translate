The IDV can display ESRI polygon shapefiles.
  1. Clear all data and displays.
  2. Load a shapefile. We'll use a catalog of shapefiles derived from the Census Tiger database provided by ESRI.
    • Open the Data Source Chooser window.
    • Click on the Catalogs tab.
    • Go to the [ht::command Main IDV Catalog]
    • Select [ht::dataset Tiger Shapefiles]->[ht::dataset Colorado]->[ht::dataset Roads]->[ht::dataset Boulder]
    • Press [ht::button Add Source]
    • The shapefile will be loaded and a [ht::display Shapefile Display] control will automatically be created.
  3. Add a background image with the [ht::menu Displays {Maps and Backgrounds} {Add Background Image}] menu item.
  4. Note the shapefile display is hidden. Raise it with the [ht::command Vertical Position] slider.
  5. This shapefile has an attribute database shown in the [ht::command Table] tab.
    • Select the [ht::command Table] tab.
    • Press the [ht::command Select Fields to Show] button to subset the fields.
    • Select all [ht::command Current Fields] by clicking in the list and pressing Control-A.
    • Press the [ht::command Remove] button.
    • In [ht::command All Fields] select [ht::param FENAME] (feature name)
    • Press the [ht::command Add] button.
    • Press [ht::button OK]
  6. Let's just show Highway 119
    • Select the [ht::command Filters] tab.
    • Under [ht::command Property] column select [ht::param FENAME]
    • Select [ht::qt Contains String]
    • Under [ht::command Value] column enter [ht::qt 119]
    • Press [ht::button Apply Filters]
