Gridded data can be accessed in the IDV from a remote server or from local files. For the exercises in this section, we will use ETA model output from the COMET Case Study on Bow Echoes (39) from June 29, 1998.
  1. [ht::startIDV]
  2. Load in the 12 Z ETA model run from June 29, 1998.
    • Open the Data Choosers window.
    • Click on the Catalogs node in the chooser tree.
    • From the catalog selector drop-down, select the IDV workshop catalog that is included in the plugin:

        /ucar/unidata/apps/workshop/workshopCatalog.xml

      If it is not in the list, type it into the selector area and press the Update button. [ht::cutAndPasteTip]

      A tree view of the data catalog will be displayed in the window below the selector.

    • Expand the Data for Comet Case Study 039 tab, and then the NCEP Model Data tab.
    • Select the ETA 1998-06-29 12:00 GMT entry, and then press the Add Source button.
