The IDV has lightning data display capability.
  1. [ht::clearDisplaysAndData]
  2. Load some lighting data
    • [ht::openDataSourceChooser]
    • In the [ht::dialog Data Chooser] tree click on the [ht::command Files] node.
    • Go to the point data directory: [ht::filename <%workshop.datadir%>/lightning/].
    • Select both [ht::filename 201118719.ingest] and [ht::filename 201118720.ingest]
    • Under the [ht::command Data Source Type] pulldown menu, select "Netcdf Point Data files"
    • Press [ht::button Add Source]
  3. Create the display
    • In the [ht::command Data Sources] panel right click the lightning data source.
    • Under [ht::command Properties] tab in the Time Binning section, change Bin Size to 5 minutes
    • Click on the [ht::tab Spatial Subset] tab. Select a region encompassing South Carolinas state and press [ht::button OK].
    • In the [ht::command Fields] panel select [ht::param Point Data].
    • In the [ht::command Displays] select [ht::display Point Data Plot] and press [ht::button Create Display]
    • Select [ht::command Layout Model] tab
    • Click on the menu and select Lightning > Bolts
    • In the same menu click Edit
    • Double click on the lightning layout model
    • Ensure that the parameter being displayed is "sgnl", and Press [ht::button OK].
    • Press [ht::button Create Display]
  4. Improving the lightning display in the [ht::command Displays] panel
    • Disable [ht::command Declutter].
    • In the [ht::command Times] panel, select [ht::command {Show Multiple}]
    • Select the button to show the [ht::command Time Settings] dialog
    • For Start Time choose [ht::button Relative to End Time], and click on button to the right of that dropdown
    • In the [ht::command Set Start Offset] Dialog select minus (-) 20 minutes.
    • For End Time choose [ht::button From Animation Time], and Press [ht::button OK]
  5. Turn on animation
