The transect view offers a 2D vertical transect.
  1. Create a Transect and Map view window under [ht::button Misc] in the [ht::button New Window] tab in the Quicklinks. (Or from the menu: [ht::menu File New {View Window} Misc {Transect and Map}]).
  2. Close the original main Map View.
  3. [ht::loadBasicGrid]
  4. Click in the left Transect View to select it and create a Color Shaded Vertical Cross Section.
  5. Change some of the state of the Transect:
    • Change the vertical scale with the [ht::menu View {Vertical Scale}] menu.
    • Toggle on/off the Grid Lines and Display Scales in under the [ht::menu View Show] menu.
    • Zoom and pan in the transect.
  6. Make sure the left Transect View is selected. Select [ht::menu {3D Grid} Derived {Flow Vectors}] in the Field Selector and create a Wind Barb Cross Section.
  7. Where are we looking at?
