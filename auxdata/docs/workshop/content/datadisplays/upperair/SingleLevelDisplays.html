The IDV can plot upper level maps as point displays from the RAOB data.

  1. Open the RAOB Single Level Data Source Chooser
    • [ht::openDataSourceChooser]
    • In the [ht::dialog Data Chooser] tree click on the [ht::command Point] node.
    • Click on the [ht::command {Upper Air}] tab.
    • In the [ht::command Server:] entry box, select the remote ADDE server [ht::command adde.ucar.edu]. In the [ht::command Data Type:] entry box, select the [ht::command Upper Air Data] entry.
    • Click on [ht::button Connect].
    • The IDV will connect to the ADDE server at adde.ucar.edu and download metadata about available RAOB data. You can load data in by relative times, or by absolute times. Select the [ht::command Relative times] radio button and the [ht::param {4 most recent}] times option.
    • From the [ht::command Level] list, select [ht::command 500] for the 500 millibar level.
    • Click the [ht::command Add Source] button.

  2. If the [ht::command {Create display}] option was checked in the chooser, a [ht::display {Point Data Plot}] will automatically be created. If not, create one.
    • In the [ht::dialog Field Selector] window create a [ht::display Point Data Plot] display using the [ht::param RAOB Point Data] field.

    3. The point data plot appears in the main window.

    [ht::cimg images/SingleLevelPlot.gif]

  3. Set the projection to the continental U.S. ([ht::menu Projections Predefined US CONUS] menu).
  4. Use the [ht::display Point Data Plot] control to change the plot density to show more stations.
  5. Remove the display
  6. Now, plot data at a different level and region.
    • In the [ht::dialog Field Selector] window select a different level (e.g. 250 mbar) in the [ht::tab Level] tab of the [ht::command Data Subset] panel.
    • Under the [ht::tab Region] tab, select a region of interest.
    • Click the [ht::button Create Display] button.
