Come up with an interesting ISL file. Here are some ideas:
  1. Do some different image manipulations. Add in your own logo or text annotation.
  2. Create a bundle with real time radar. Loop 10 times, sleeping 10 minutes each loop. Call [gen::islTag reload] to reload the data. Capture an image. Use the date/time properties in your filename.
  3. Try to create a procedure that loads a bundle, and writes an image. Call it with a set of bundles from a [gen::islTag foreach] loop tag.
