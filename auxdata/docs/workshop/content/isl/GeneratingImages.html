Load in a bundle and generate a screen shot.
  1. First of all save this isl.xidv file to your isl directory.
  2. Run this file: [gen::isl image1.isl]
  3. Now, change directory to the home directory (cd) and run: 
    runIDV <%workshop.homedir%>/isl/image1.isl
  4. What happened?
  5. The bundle reference in the isl was relative to where you run the idv from. We'll use the islpath property: [gen::isl image2.isl]
  6. Note, no images were created. Only the bundle was loaded. To create an image we'll use the [gen::islTag image] tag: [gen::isl image3.isl]
  7. The [gen::islTag movie] tag lets us create a time-loop movie or animated gif. [gen::isl image4.isl]
  8. The IDV gives you an easy way to save off a basic ISL file. Run the IDV with the above bundles. Select [ht::menu File {Save As...}] and choose the [ht::filename <%workshop.homedir%>/isl] directory and enter [ht::filename image.isl].
  9. At the ISL Properties dialog:
    1. Turn off Offscreen.
    2. Turn off Image
    3. Turn on Movie and enter [ht::filename image.kmz].
    4. Press OK. This saves a bundle (image.xidv) and an ISL file.
  10. The ISL file should look like this: [gen::isl image.isl]
  11. If we enter a [ht::filename kmz] suffix for the movie the IDV will generate a Google Earth kmz file of the animation loop.
  12. Now, we'll show you an easier way to do all of this.
    • Bring up the IDV again and load the above bundle.
    • Now, do a [ht::menu File {Save As}] and this time call it [ht::filename idv.isl].
    • Select [ht::button Generate a Movie] and press [ht::button OK].
    • Let's look at this file.
    • Now, save the bundle as idv2.isl and this time select [ht::command Include Bundle Inline] and press [ht::button OK].
