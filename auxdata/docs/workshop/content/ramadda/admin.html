Creating the Admin Account

Once you have installed RAMADDA, you should be at this screen, and ready to set up the admin account.

  1. Click "Next".
  2. Read and accept the license agreement, and click "Next".
  3. You will now be prompted for admin credentials after which you will click "Initialize Server".
    4. Be sure to leave "Install all plugins" checked.
  4. You should now be at this screen. Please login with your admin account.

You have now set up the RAMADDA admin account and you are ready to set up users and roles.
