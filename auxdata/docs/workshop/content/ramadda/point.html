Working with Point Data

RAMADDA can also work with point data observations

  1. Go to your RAMADDA Instance and navigate to your "Users" folder.
  2. Create a new entry by clicking the plus sign. You will have a panoply of choices. Select "Tá»‡p".
  3. Upload the file /data/idv/point/earthquake.nc
  4. You can examine the netCDF metadata by going to [ht::menu Data {File Metadata}]
  5. Select [ht::menu Data {Plot Points on a Map}]
  6. Select [ht::menu Data {CSV, KML Output}]
  7. Select a format and subset over a region.
