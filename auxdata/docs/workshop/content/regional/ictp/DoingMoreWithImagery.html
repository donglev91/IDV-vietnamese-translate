Try these exercises on your own.
  1. Load in images over Africa from adde.ucar.edu using the GLOBAL dataset.
  2. If you have access to multi-banded imagery, you can create a true color image from 3 channels that would represent the Red, Green and Blue bands of the imagery.
    • Use the satellite chooser to load in the data.
      1. Choose adde.eumetsat.int as the server and EUM_AD as the dataset.
      2. When prompted, enter in the user and project information given by the intructor.
      3. Select the [ht::command M9] dataset.
      4. Uncheck the [ht::command Create display] checkbox.
      5. Set the [ht::command Data Type] to [ht::param Brightness] and the [ht::command Channel] to [ht::param All Bands].
      6. Use the first [ht::command Magnification] slider to set the magnification to -4
      7. Click the [ht::button Add Source] button.
    • In the [ht::dialog Field Selector], select the [ht::dataset Formulas] data source and use the [ht::param {Imagery->Three Color (RGB) Image}] formula. Select the [ht::display {3 Color (RGB) Image}] display, then click the [ht::button Create Display] button.
    • When prompted for the bands, pick the approapriate field for the red, green and blue parameters (1.6, 0.8, 0.6 microns) and click [ht::command OK].
