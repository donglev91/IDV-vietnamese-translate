In this section we will cover how to automatically capture QuickTime movies from the command line.
  1. To automatically capture a QuickTime movie from the command line we will use the [ht::filename movie.xidv] bundle we created previously.
  2. Exit the IDV if it is running and run the following command: 
        runIDV -movie test.mov <%workshop.homedir%>/movie.xidv
  3. You should see the IDV start up, load in the bundle and then exit.
  4. You should have the test.mov file created in the local directory.
