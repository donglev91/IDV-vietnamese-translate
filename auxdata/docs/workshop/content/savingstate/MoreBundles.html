Let's do more with bundles.
  1. Let's only save loaded data sources
    1. If you don't have any data and displays, load the [ht::filename workshop.xidv] bundle with the [ht::menu File {Open}] menu.
    2. Let's also load the Sample AVN Global grid data from the Catalogs tab.
    3. Select the [ht::menu File {Save As}] menu item.
    4. Enter [ht::filename justdata.xidv] as a file name.
    5. Uncheck [ht::button Views] and [ht::button Displays] and click [ht::button Save].
    6. Open this bundle with the menu [ht::menu File {Open}] and check remove all displays and data.
  2. Let's only save the displays.
    1. From the field selector, create a Color-Shaded Plan View from the [ht::param temperature] field from the AVN Data Source.
    2. Save the bundle with the menu [ht::menu File {Save As}].
    3. Enter [ht::filename justdisplays.xidv] as a file.
    4. Uncheck [ht::button Views] and [ht::button Data Sources].
    5. Check [ht::button Displays].
    6. Click [ht::button Save].
    7. Remove the displays with the [ht::menu Edit {Remove all displays}] menu item.
    8. Open the [ht::filename justdisplays.xidv] bundle with the menu [ht::menu File {Open}].
    9. Make sure you UNCHECK "[ht::button Remove displays and data.]"
    10. When we load in a bundle with only displays the IDV will prompt you for the data.
    11. Choose the [ht::param temperature] field from the AVN data source.
