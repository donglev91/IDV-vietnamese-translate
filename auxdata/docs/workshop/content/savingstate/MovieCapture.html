In this section we will cover how to create QuickTime movies, animated gifs and Goole Earth files within the IDV.
  1. First, load in some data with multiple time steps and create a display.
    • Remove any data and displays. [ht::refresh]
    • [ht::loadEtaGrid]
    • [ht::createBasicDisplay]
  2. Save this state off as the [ht::qt movie] bundle.
    • Select the [ht::menu File {Save As...}] menu item.
    • Make sure you are in your home directory and enter [ht::filename movie.xidv].
  3. Open the [ht::dialog Movie Capture] window by selecting the [ht::menu View Capture {Movie}] menu item from the map view's menu bar.
  4. For now the IDV requires that the window you are capturing an image in is not obscured. Move any other windows out of the way.
  5. A movie is simply a sequence of frames. There are three ways to capture this sequence.
  6. You can capture one frame with the [ht::button Capture one image] button.
    • Try 3 or 4 cycles of capturing one image and moving the viewpoint.
    • You should see the [ht::command Frames] panel become enabled and the number of frames listed.
    • Bring up the [ht::dialog Movie Preview] window by clicking the [ht::button Preview] button.
    • In the [ht::dialog Movie Preview] window you can step forward, backward and play the movie as well as delete individual frames.
  7. Start fresh by clicking the [ht::button Delete all] button in the main [ht::dialog Movie Capture] window.
  8. Now, let's capture an entire animation by clicking the [ht::button Capture Time Animation] button.
  9. When the animation is complete the [ht::dialog File Dialog] will automatically be shown. [ht::save workshop.mov]
  10. The [ht::button Capture automatically] button allows you to capture a single frame every N seconds.
    • Click the [ht::button Capture automatically] button to start the capture.
    • Turn on animation.
    • Navigate around the 3D view. Zooming in, etc.
    • Click the [ht::button Stop] button and then click [ht::button Preview].
    • The [ht::dialog Preview] window allows you to view, play and delete the frames that make up a movie.
  11. The IDV treats QuickTime movies as yet another data source. Load in and view the newly created movie by:
    • [ht::openDataSourceChooser]
    • Go to the [ht::command Files] tab.
    • Select [ht::filename workshop.mov].
    • For Quicktime movies the data source is not shown and the [ht::display Movie Display Control] will automatically be created.
  12. You can also save the movie as a Google Earth kmz file.
    • Press the [ht::button Delete All] button.
    • Capture the animation with the [ht::button Capture animation] button.
    • When done, choose [ht::filename movie.kmz] as the file.
    • Bring this file up in Google Earth.
