[ht::desc The IDV can publish content to a RAMADDA server]
In this section we will cover how the IDV can publish images, movies, bundles and data to a RAMADDA server. RAMADDA is a content management system
  • First we need to install the RAMADDA Publisher plugin.
    • Go to the [ht::menu Tools {Plugin Manager}] menu to bring up the Plugin Manager.
    • Under Miscellaneous install the RAMADDA Publisher
    • Shut down and restart the IDV.
  • Create a publisher account.
    • Go to the [ht::menu File Publish New {RAMADDA Publisher}] menu.
    • Enter:
      Name: Workshop account
      Server: motherlode.ucar.edu
      User Name: workshop
      Password: unidata
    • Press OK.
  • Now, any of the file dialogs for saving images, movies, data or bundles will have a "Select Publisher" menu.
  • Publish an image and bundle to RAMADDA
    • Load up some data and create a display.
    • Save an image off under the [ht::menu View Capture Image] menu.
    • In the File dialog enter a image file name and also select the "Workshop account" to publish to.
    • In the Publisher dialog enter a name, description and select [ht::menu Top {IDV Community Resources} {Workshop Sandbox}]
  • In the Catalog chooser go to [ht::menu {Unidata's RAMADDA Server} {IDV Community Resources} {Workshop Sandbox}]. Note - If you right click on the entry you can Load Remote Catalog and then do a refresh to update for any changes.
  • Select one of the bundles and load it into the IDV.
