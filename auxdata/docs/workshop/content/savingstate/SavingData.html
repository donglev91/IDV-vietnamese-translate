[ht::desc The IDV can save both point and gridded netcdf files]
In this section, we will cover how to write out point and gridded data files.
  1. First, clear all existing data and displays and load in the Sample ETA Grid from the catalog chooser.
  2. Create a Contour Plan View of Temperature @ 500 MB.
  3. Save the displayed data.
    • In the Dashboard Displays tab, select the [ht::menu File {Save} {Export Displayed Data to NetCDF...}] menu from the display's File menu.
    • Enter [ht::filename testgrid.nc].
  4. Now, let's load this grid back into the IDV. Note the parameter differences. Create another Contour Plan View with T. [ht::menu Data {Choose Data} {From the File System}]
  5. Let's save a derived quantity.
    • Choose Geostrophic Wind Vectors (from Z) under 3D Grid - Derived and create a Vector Plan View.
    • Repeat the above steps, saving the file as testgrid2.nc
  6. Let's save point data
    • Load in a couple of time steps of real time METARS from the ADDE Point Data chooser.
    • Create a Station Model display of the data.
    • Write out the NetCDF file with the [ht::menu File Save {Export to NetCDF}] menu.
